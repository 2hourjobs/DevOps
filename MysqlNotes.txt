## Mysql Backup and Restore command
$ mysqldump -u root --host 172.17.0.2 -p crawl > crawl0-bak.sql # this command will export crawl database as name crawl0-bak.sql
$ mysql -u username --host 172.17.0.2 -p crawl < crawl0-bak.sql # this command will restore the crawl database from crawl0-bak.sql backup file
> source ~/Desktop/backup.sql # inside mysql server this command will import the sql file and restore database.
